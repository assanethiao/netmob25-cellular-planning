ğŸ”¹ Netmob25 Mobility Model

Netmob25 is a realistic urban mobility model based on datasets from the Netmob Data Challenge.

Unlike simple mobility models (Random Walk, Random Waypoint), Netmob25 reproduces:

Daily commuting patterns

Urban hotspots

Peak hours

Spatial clustering

Realistic speed distributions

It generates time-based mobility traces:

user_id | time | x | y | speed


These traces are imported into NS-3 to move user nodes.


ğŸ”¹ NS-3 Network Simulator

In this project, NS-3 is used to:

Model radio propagation

Perform handovers

Simulate TCP/UDP traffic

Measure performance metrics
